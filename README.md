# BC y Copilot

Blog personal sobre Microsoft Dynamics 365 Business Central, GitHub Copilot, automatizacion y productividad tecnica.

## Desarrollo local

Requisitos:

- Ruby
- Bundler

Comandos principales:

```powershell
bundle install
bash tools/run.sh
```

Build de validacion:

```powershell
bash tools/test.sh
```
